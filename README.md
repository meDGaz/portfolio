# Mohamed Gazzoun - Portfolio

Bienvenue sur le dépôt de mon portfolio professionnel. Ce projet est une Single Page Application (SPA) développée en HTML, CSS et JavaScript Vanilla, conçue pour présenter mes compétences de Développeur Fullstack (Laravel & Vue.js).

## 🚀 Technologies utilisées

- **HTML5** & **CSS3** (Variables CSS, Grid, Flexbox, Animations)
- **JavaScript (Vanilla)** pour les effets interactifs (Matrice, effets 3D Tilt, scroll)
- Aucun framework lourd : L'objectif était de démontrer une maîtrise des fondamentaux du Web pour garantir des performances optimales.

## ✨ Fonctionnalités

- **Design "Bento Grid" Moderne** : Interface épurée et structurée en cartes.
- **Effets Graphiques** :
  - Arrière-plan dynamique et subtil.
  - Effet "Matrix" derrière la photo de profil.
  - Cartes interactives (Tilt 3D) au survol.
  - Glitch effect sur le titre principal.
- **Entièrement Responsive** : S'adapte parfaitement à tous les écrans, du mobile aux écrans larges.
- **Dark Mode Natif** : L'interface repose sur une palette de couleurs sombres et professionnelles (teintes *Slate* et accents néon).
- **Hébergé via GitHub Pages** : Déploiement statique simple et ultra-rapide sans besoin de serveur backend (Node, PHP, etc).

## 🛠 Installation / Déploiement Local

Pour lancer ce projet localement et explorer le code :

1. Clonez ce dépôt :
   ```bash
   git clone https://github.com/votre-nom-d-utilisateur/votre-repo.git
   ```
2. Ouvrez le dossier du projet :
   ```bash
   cd votre-repo
   ```
3. Lancez le fichier `index.html` :
   - Ouvrez simplement `index.html` depuis votre navigateur.
   - Ou utilisez une extension comme **Live Server** sur VS Code pour une expérience avec rafraîchissement automatique.
   - En Python : 
     ```bash
     python -m http.server 8000
     ```
